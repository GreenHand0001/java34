# 会员管理系统 java34

## 项目概述

本会员管理系统采用前后端分离的设计模式，基于Spring Boot、Vue.js以及Redis等先进技术构建。系统功能全面，涵盖了会员管理、订单处理、产品管理等多个业务模块。

## 技术栈

- 后端：Spring Boot、Spring、Spring MVC、MyBatis
- 前端：Vue.js
- 缓存：Redis

## 功能模块

- **首页统计**
- **产品管理**
  - 产品分类
  - 产品中心
- **会员管理**
  - 会员
  - 会员等级
  - 等级任务
  - 用户账单
  - 在线会员
- **订单管理**
  - 订单
  - 物流快递
- **电子券管理**
  - 优惠券制作
  - 已发布优惠券
  - 优惠券领取记录
- **积分管理**
- **系统管理**
  - 用户管理
  - 菜单管理
- **系统监控**
  - 操作日志
  - 异常日志
- **系统工具**
  - 定时任务
  - 存储管理
  - 邮件工具

## 运行环境

- JDK 1.8
- Redis 3.0
- MySQL 8.0

- Node.js 14.15
